## Full Description

### English

PixelBridge makes it effortless to install the AppLovin Axon Pixel on your Shopline store — no coding required.

**Why Axon Pixel?**
AppLovin's Axon Pixel enables precise ad attribution, helping you measure and optimize your Return on Ad Spend (ROAS) across AppLovin's advertising network.

**Key Features:**
- One-click installation — pixel is auto-injected into your store
- Tracks all key e-commerce events: page views, add-to-cart, checkout, and purchases
- Real-time event statistics dashboard
- SHA-256 hashed user data for privacy compliance
- Purchase deduplication to ensure accurate attribution
- Multi-layer add-to-cart detection (fetch/XHR interception, Shopline Analytics API, click listeners)
- Zero impact on store performance (async loading)

**How it works:**
1. Install the app and enter your AppLovin Axon Pixel Key
2. The pixel is automatically added to every page of your store
3. E-commerce events are tracked and sent to AppLovin for attribution
4. View aggregate statistics in your dashboard

**Pricing:**
$1.99/month with a 3-day free trial. Cancel anytime.
